# Лабораторная работа №2 — Профиль пользователя (Flutter)

## О проекте

Приложение "Мой профиль" с четырьмя вкладками через `BottomNavigationBar`:

- **Профиль** — аватар, контакты, интересы, кнопки
- **Галерея** — сетка изображений (`GridView`)
- **Контакты** — список из 5 контактов (`ListView.builder`)
- **О приложении** — информация о лабораторной

### Реализованные возможности

- Переключение тёмной и светлой темы
- Интерактивный счётчик лайков
- SnackBar при нажатии "Написать"
- Современный Material 3 дизайн

## Использованные виджеты

- `BottomNavigationBar` + 4 вкладки
- `GridView.count` + `Image.network`
- `ListView.builder`
- `StatefulWidget` + `setState`
- `Card`, `ListTile`, `CircleAvatar`, `Chip`, `Wrap`
- `Navigator`, `SnackBar`

## Как запустить

1. Открыть **FlutLab.io**
2. Создать новый проект
3. Заменить содержимое `lib/main.dart` на код из этого репозитория
4. Нажать **Build**
